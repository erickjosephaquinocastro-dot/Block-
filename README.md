# Blog Personal con Autenticación Básica

Proyecto sencillo de Blog usando HTML, CSS y JavaScript. Incluye:

- Registro e inicio de sesión con LocalStorage.
- Perfil de usuario con avatar, nombre visible y biografía.
- Cierre de sesión.
- CRUD de publicaciones: crear, leer, editar y eliminar.
- Persistencia local en el navegador.
- Interfaz responsiva y ventanas modales interactivas.

## Uso

1. Abre `index.html` en tu navegador.
2. Regístrate con un usuario y contraseña.
3. Crea, edita y elimina publicaciones.

## Estructura

- `index.html` — Interfaz del blog.
- `styles.css` — Estilos responsivos.
- `script.js` — Lógica de autenticación, perfil y CRUD.
- `data.json` — Ejemplo de estructura JSON de usuarios y publicaciones.

## GitHub Pages

Para publicar en GitHub Pages, crea un repositorio con estos archivos y habilita Pages desde la rama `main`.
